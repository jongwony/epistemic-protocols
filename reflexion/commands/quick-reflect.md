# /quick-reflect

Generate a quick summary of this session without saving to memory.

Runs only Phase 1-2 of reflexion:
1. Detect session context
2. Generate summary and extract insights

Does NOT:
- Ask for insight selection
- Save anything to memory
- Modify any files

Use this for quick review before ending a session.
